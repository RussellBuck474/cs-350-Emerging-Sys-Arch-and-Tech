# cs-350-Emerging-Sys-Arch-and-Tech
Repository for Embedded Systems Class
This repository shows utilizing a Raspberry Pi to put together a Smart thermostat. All the steps and programming is included.
Project Summary
The overall goal of these projects was to design and implement solutions that integrate software with hardware components while considering system architecture and performance. The problem being solved involved creating efficient, reliable, and scalable systems that could interact with hardware while meeting specific application requirements.
Through these projects, I explored how hardware constraints, processing efficiency, and architectural design decisions affect system performance. I also evaluated different technologies and approaches to determine the most appropriate solution for each scenario.

What I Did Well
One of my strengths in this project was my ability to:

Design clear and functional interfaces between software and hardware components
Apply logical problem-solving to troubleshoot and refine system behavior
Break complex problems into manageable steps
Document my code and design decisions effectively

I also demonstrated strong analytical thinking when evaluating how different architectural choices impacted system efficiency and performance.
